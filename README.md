# E-Commerce Website

A Ecommerce Website made with React.js Framework.



## Run Locally

Clone the project

```bash
  [(https://github.com/Yashas-DR/QuickCraft_frontend.git)]
```

Go to the project directory

```bash
  cd React_E-Commerce
```

Install dependencies

```bash
  npm install
```
```bash
  npm install axios
```
Start the server

```bash
  npm start
```
